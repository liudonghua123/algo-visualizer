# 排序算法可视化应用 - 产品需求文档

## 1. 产品概述

一个交互式的排序算法可视化教育工具，通过动态演示帮助用户直观理解各种排序算法的工作原理。应用程序将展示不同排序算法（如冒泡排序、快速排序、归并排序等）如何逐步处理数据，并以流畅的动画效果呈现比较、交换和已完成状态。

### 核心价值
- 降低算法学习的认知门槛
- 通过可视化增强对排序过程的理解
- 支持自定义配置以满足不同学习需求

### 目标用户
- 计算机科学学生
- 编程初学者
- 需要复习排序算法的开发者
- 对算法感兴趣的教育工作者

## 2. 核心功能

### 2.1 算法模块
支持的排序算法：
1. **冒泡排序（Bubble Sort）** - 经典比较排序算法
2. **选择排序（Selection Sort）** - 原地比较排序
3. **插入排序（Insertion Sort）** - 类扑克牌排序
4. **快速排序（Quick Sort）** - 分治策略排序
5. **归并排序（Merge Sort）** - 分治合并排序
6. **堆排序（Heap Sort）** - 基于堆结构的排序

### 2.2 控制功能
- **播放/暂停** - 控制排序过程的开始和暂停
- **重置** - 重新生成随机数据
- **单步执行** - 逐步观察排序过程
- **速度调节** - 控制动画速度（慢速/中速/快速）
- **数组大小调整** - 调整待排序元素数量

### 2.3 视觉反馈
- **比较状态** - 高亮显示当前正在比较的元素
- **交换状态** - 动画显示元素位置交换
- **已完成状态** - 已排序好的元素标记为完成
- **枢轴状态（快速排序）** - 标记当前枢轴元素

## 3. 用户界面设计

### 3.1 整体风格
- **设计理念**: 深色科技风格，营造专业的编程学习氛围
- **背景**: 深灰渐变背景，配合微妙的网格线纹理
- **主色调**: 深色背景 + 霓虹蓝/紫色作为强调色
- **字体**: 等宽字体显示数值，科技感强的无衬线字体用于标题

### 3.2 颜色方案
| 状态 | 颜色代码 | 用途 |
|------|----------|------|
| 默认条形 | #6366f1 (靛蓝色) | 未排序/待处理元素 |
| 比较中 | #f59e0b (琥珀色) | 当前正在比较的元素 |
| 交换中 | #ec4899 (粉红色) | 正在交换位置的元素 |
| 已完成 | #10b981 (翠绿色) | 已确定顺序的元素 |
| 背景色 | #0f172a (深蓝灰) | 主背景 |
| 次级背景 | #1e293b (中灰蓝) | 卡片/面板背景 |

- **桌面端**: 完整布局，所有控制项可见

- **移动端**: 垂直布局，控制按钮增大触控区域

## 4. 动画与交互设计

### 4.1 柱状图动画
- 高度代表数值大小
- 宽度根据数组数量动态调整
- 使用 CSS transition 实现平滑高度变化
- 交换动画：左右平移效果

### 4.2 状态转换
- 比较 → 默认: 快速闪烁过渡
- 交换 → 默认: 平滑渐变消失
- 默认 → 已完成: 颜色渐变加深

### 4.3 用户交互
- 按钮悬停: 发光效果 + 轻微放大
- 按钮点击: 缩放反馈动画
- 滑块拖动: 实时更新数值显示

## 5. 技术实现要求

### 5.1 性能要求
- 支持至少100个元素的流畅动画
- 动画帧率保持60fps
- 避免布局抖动（Layout Thrashing）

### 5.2 可访问性
- 所有交互元素可通过键盘操作
- 颜色对比度符合WCAG标准
- 支持屏幕阅读器

### 5.3 技术栈
- React 18+ (状态管理和组件化)
- Vite (构建工具)
- 纯 CSS 动画 (性能优化)
- TypeScript (类型安全)
