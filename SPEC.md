# CodeVault - 全能代码编辑器/查看器规范文档

## 1. 项目概述

**项目名称：** CodeVault  
**项目类型：** Flutter 跨平台应用  
**核心功能：** 一个功能完善的文本/代码编辑器，支持语法高亮、代码折叠、跳转等功能，可在桌面端、移动端和Web端运行。

**目标用户：** 开发者、学生、技术写作者等需要查看和编辑代码文件的人群。

---

## 2. 技术栈与框架

### 2.1 核心框架
- **Flutter SDK:** 主框架（用户已安装在 `C:\Users\admin\apps\flutter`）

## 3. 功能需求

### 3.1 文件管理功能
- ✅ 打开本地文件（通过文件选择器）
- ✅ 创建新文件
- ✅ 保存文件（保存到原路径或另存为）
- ✅ 最近打开的文件列表
- ✅ 文件浏览器（可选）

### 3.2 编辑器核心功能
- ✅ **语法高亮：** 支持100+编程语言的语法高亮
- ✅ **代码折叠：** 支持基于缩进或语法的代码折叠
- ✅ **行号显示：** 显示行号并支持点击跳转
- ✅ **跳转到行：** 支持跳转到指定行号
- ✅ **搜索和替换：** 在文件中搜索和替换文本
- ✅ **撤销/重做：** 支持撤销和重做操作
- ✅ **复制/剪切/粘贴：** 标准编辑操作
- ✅ **自动保存：** 可选的自动保存功能

### 3.3 支持的文件类型
默认支持：
- 编程语言：JavaScript, TypeScript, Python, Java, C, C++, C#, Go, Rust, PHP, Ruby, Swift, Kotlin, Dart 等
- 标记语言：HTML, CSS, SCSS, JSON, XML, YAML, Markdown
- 配置文件：.env, .gitignore, .properties, .toml, .ini
- 其他文本：.txt, .log, .csv

**可扩展性：** 用户可在设置中添加自定义文件类型映射

### 3.4 主题系统
- ✅ **亮色主题：** 清晰的浅色界面
- ✅ **暗黑主题：** 适合夜间编程
- ✅ **代码高亮主题：** 支持多种代码高亮配色方案
  - One Dark Pro
  - Monokai
  - GitHub
  - Dracula
  - Solarized Light/Dark
- ✅ **主题跟随系统：** 自动跟随系统主题

### 3.5 响应式设计
- ✅ **桌面端：** 完整的侧边栏、菜单栏、键盘快捷键
- ✅ **移动端：** 触摸优化、简洁界面、底部工具栏
- ✅ **Web端：** 浏览器适配、拖拽上传

### 3.6 设置功能
- ✅ 主题选择（亮色/暗黑/跟随系统）
- ✅ 字体大小调节
- ✅ 字体家族选择
- ✅ 代码高亮主题选择
- ✅ Tab宽度设置
- ✅ 自动保存开关
- ✅ 自定义文件类型映射
- ✅ 语言设置

## 6. GitHub Actions 自动化

### 6.1 工作流设计
```yaml
# .github/workflows/
build.yml      # 构建所有平台
release.yml   # 发布到 GitHub Release
pages.yml     # 部署到 GitHub Pages
```

### 6.2 构建目标
- **Windows:** `.exe` 便携版
- **macOS:** `.app` + `.dmg`
- **Linux:** `.AppImage` + `.deb`
- **Android:** `.apk` + `.aab`
- **iOS:** `.ipa` (需要证书)
- **Web:** 静态资源包

### 6.3 发布流程
1. 创建 tag 触发构建
2. 编译所有平台
3. 上传到 Release Assets
4. 部署 Web 到 gh-pages
