# 算法可视化平台

一个交互式的算法和数据中心可视化教育工具，通过动态演示帮助用户直观理解各种算法和数据结构的工作原理。

## 功能特性

### 📊 排序算法
- 冒泡排序
- 选择排序
- 插入排序
- 快速排序
- 归并排序
- 堆排序

### 🔍 查找算法
- 线性查找
- 二分查找

### 🌳 树结构
- 二叉树遍历（前序、中序、后序）

### 🕸️ 图算法
- 广度优先搜索 (BFS)
- 深度优先搜索 (DFS)
- Dijkstra 最短路径

### 📦 数据结构
- 栈
- 队列
- 链表

## 技术栈

- React 18
- TypeScript
- Vite
- React Router
- Tailwind CSS

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

推送到 GitHub 后，GitHub Actions 会自动构建并部署到 GitHub Pages。
